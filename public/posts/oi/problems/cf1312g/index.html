CF1312G-Autocompletion

题目:

题目描述:

You are given a set of strings S S S . Each string consists of lowercase Latin letters.

For each string in this set, you want to calculate the minimum number of seconds required to type this string. To type a string, you have to start with an empty string and transform it into the string you want to type using the following actions:

  • if the current string is t t t , choose some lowercase Latin letter c c c and append it to the back of t t t , so the current string becomes t+c t + c t+c . This action takes 1 1 1 second;
  • use autocompletion. When you try to autocomplete the current string t t t , a list of all strings s∈S s \in S s∈S such that t t t is a prefix of s s s is shown to you. This list includes t t t itself, if t t t is a string from S S S , and the strings are ordered lexicographically. You can transform t t t into the i i i -th string from this list in i i i seconds. Note that you may choose any string from this list you want, it is not necessarily the string you are trying to type.

What is the minimum number of seconds that you have to spend to type each string from S S S ?

Note that the strings from S S S are given in an unusual way.

输入格式:

The first line contains one integer n n n ( 1≤n≤106 1 \le n \le 10^6 1≤n≤106 ).

Then n n n lines follow, the i i i -th line contains one integer pi p_i pi​ ( 0≤pi<i 0 \le p_i < i 0≤pi​<i ) and one lowercase Latin character ci c_i ci​ . These lines form some set of strings such that S S S is its subset as follows: there are n+1 n + 1 n+1 strings, numbered from 0 0 0 to n n n ; the 0 0 0 -th string is an empty string, and the i i i -th string ( i≥1 i \ge 1 i≥1 ) is the result of appending the character ci c_i ci​ to the string pi p_i pi​ . It is guaranteed that all these strings are distinct.

The next line contains one integer k k k ( 1≤k≤n 1 \le k \le n 1≤k≤n ) — the number of strings in S S S .

The last line contains k k k integers a1 a_1 a1​ , a2 a_2 a2​ , …, ak a_k ak​ ( 1≤ai≤n 1 \le a_i \le n 1≤ai​≤n , all ai a_i ai​ are pairwise distinct) denoting the indices of the strings generated by above-mentioned process that form the set S S S — formally, if we denote the i i i -th generated string as si s_i si​ , then S=sa1,sa2,…,sak S = {s_{a_1}, s_{a_2}, \dots, s_{a_k}} S=sa1​​,sa2​​,…,sak​​ .

输出格式:

Print k k k integers, the i i i -th of them should be equal to the minimum number of seconds required to type the string sai s_{a_i} sai​​ .

思路:

发现给出的串格式其实就是一个 trietrietrie。

那把 trietrietrie 建出来之后，发现操作就可以变成这样：往自己子节点走，代价 111，以及往自己子树内第 xxx 个关键点走，代价 xxx。

然后可以类似于 dfsdfsdfs 序，只不过如果当前这个位置不在 SSS 里就不加 cntcntcnt，然后子树内的关键点 ididid 就连续了。

连续了之后我们令 dpi=dp_i =dpi​= 到节点 iii 的最少步数，很明显可以从 fafafa 转移，并且还可以从任意一个祖先转移，即 dpi=min⁡(dpanc+idi−idanc+hav_anc)dp_i = \min(dp_{anc} + id_i - id_{anc} + hav\_{anc})dpi​=min(dpanc​+idi​−idanc​+hav_anc)，这里的 havhavhav 是指祖先是不是关键点。 然后这个移项后就变成了 dpi=min⁡(dpanc−idanc+hav_anc)+ididp_i = \min(dp_{anc} - id_{anc} + hav\_{anc})+id_idpi​=min(dpanc​−idanc​+hav_anc)+idi​​，又因为只有祖先能转移，所以就树上前缀 minminmin 优化一下即可。

#include "ybwhead/ios.h"
using namespace std;
int n;
const int maxn = 1e6 + 10;
int a[maxn];
int b[maxn];
vector<int> v[maxn];
int tot, tr[maxn][26];
int num[maxn];
int f[maxn];
int ne(int u, int v, int w)
{
    if (!tr[u][w])
        tr[u][w] = num[v] = ++tot, f[tot] = u;
    else
        num[v] = tr[u][w];
    // cout << u << " " << v << " " << w << " " << endl;
    return 0;
}
int dp[maxn];
int up[maxn];
int dfn;
void dfs(int u)
{
    dfn += a[u];
    dp[u] = min(dp[u], dp[f[u]] + 1);
    if (a[u])
    {
        dp[u] = min(dp[u], up[f[u]] + dfn);
    }
    up[u] = min(up[f[u]], dp[u] - (dfn - a[u]));
    for (int i = 0; i < 26; i++)
        if (tr[u][i])
            dfs(tr[u][i]);
}
int main()
{
    yin >> n;
    num[1] = 1;
    tot = 1;
    for (int i = 2; i <= n + 1; i++)
    {
        int x;
        char c;
        yin >> x >> c;
        ++x;
        ne(num[x], i, c - 'a');
    }
    yin >> n;
    memset(dp, 0x3f, sizeof(dp));
    memset(up, 0x3f, sizeof(up));
    dp[1] = 0;
    for (int i = 1; i <= n; i++)
    {
        yin >> b[i];
        ++b[i];
        ++a[num[b[i]]];
    }
    dfs(1);
    for (int i = 1; i <= n; i++)
        yout << dp[num[b[i]]] << " ";
    return 0;
}
