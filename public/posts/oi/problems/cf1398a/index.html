CF1398A-Bad Triangle

题目:

题目描述:

You are given an array a1,a2,…,an a_1, a_2, \dots , a_n a1​,a2​,…,an​ , which is sorted in non-decreasing order ( ai≤ai+1) a_i \le a_{i + 1}) ai​≤ai+1​) .

Find three indices i i i , j j j , k k k such that 1≤i<j<k≤n 1 \le i < j < k \le n 1≤i<j<k≤n and it is impossible to construct a non-degenerate triangle (a triangle with nonzero area) having sides equal to ai a_i ai​ , aj a_j aj​ and ak a_k ak​ (for example it is possible to construct a non-degenerate triangle with sides 3 3 3 , 4 4 4 and 5 5 5 but impossible with sides 3 3 3 , 4 4 4 and 7 7 7 ). If it is impossible to find such triple, report it.

输入格式:

The first line contains one integer t t t ( 1≤t≤1000 1 \le t \le 1000 1≤t≤1000 ) — the number of test cases.

The first line of each test case contains one integer n n n ( 3≤n≤5⋅104 3 \le n \le 5 \cdot 10^4 3≤n≤5⋅104 ) — the length of the array a a a .

The second line of each test case contains n n n integers a1,a2,…,an a_1, a_2, \dots , a_n a1​,a2​,…,an​ ( 1≤ai≤109 1 \le a_i \le 10^9 1≤ai​≤109 ; ai−1≤ai a_{i - 1} \le a_i ai−1​≤ai​ ) — the array a a a .

It is guaranteed that the sum of n n n over all test cases does not exceed 105 10^5 105 .

输出格式:

For each test case print the answer to it in one line.

If there is a triple of indices i i i , j j j , k k k ( i<j<k i < j < k i<j<k ) such that it is impossible to construct a non-degenerate triangle having sides equal to ai a_i ai​ , aj a_j aj​ and ak a_k ak​ , print that three indices in ascending order. If there are multiple answers, print any of them.

Otherwise, print -1.
